# Quick Start

Worldview is a react library for rendering 2D and 3D views using [regl](https://github.com/regl-project/regl).

Add regl-worldview to your React project:

```bash
npm install --save regl-worldview
```

If you're not using a module bundler or package manager we also have a global ("UMD") build hosted on the unpkg CDN. Simply add the following script tag to the bottom of your HTML file:

```js
<script src="https://unpkg.com/regl-worldview/dist/regl-worldview.umd.min.js" />
```

## Flow types

Flow types are supported. To use the type:

```js
import type { Position, Orientation, Scale, Color, Cube, Props, BaseProps } from 'regl-worldview'

const position: Position = { x: 1, y: 1, z: 1 };
const orientation: Orientation = {  x: 1, y: 1, z: 1, w: 1};
const scale: Scale = { x: 1, y: 1, z: 1 };
const color: Color = { r: 1, g: 1, b: 1, a: 0.5 };
const pose: Pose = { position, orientation }[]

const cube: Cube = { pose, scale, color }

```
