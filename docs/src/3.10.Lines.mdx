# Lines

`<Lines />` renders 3d lines and inherits from `<Command />`.

## Props

| Name     | Type     | Default | Description                     |
| -------- | -------- | ------- | ------------------------------- |
| children | `Line[]` | []      | array of Line markers to render |

### Line

```js
type Line = {
  pose: {
    position: { x: number, y: number, z: number },
    orientation: { x: number, y: number, z: number, q: number }
  }
  scale: {
    x: number,
    y: number,
    z: number,
  },
  color: {
    r: number, // between 0 and 1
    g: number, // between 0 and 1
    b: number, // between 0 and 1
    a: number, // between 0 and 1
  },
  points: [{ x: number, y: number, z: number }]
}
```
