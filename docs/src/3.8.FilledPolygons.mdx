# FilledPolygons

`<FilledPolygonss />` renders filled, flat polygon shapes and inherits from `<Command />`. It is used to render filled polygons.

## Props

| Name     | Type              | Default | Description                              |
| -------- | ----------------- | ------- | ---------------------------------------- |
| children | `FilledPolygon[]` | []      | array of FilledPolygon markers to render |

### FilledPolygon

```js
type FilledPolygon = {
  pose: {
    position: { x: number, y: number, z: number },
    orientation: { x: number, y: number, z: number, q: number }
  }
  scale: {
    x: number,
    y: number,
    z: number
  },
  color: {
    r: number, // between 0 and 1
    g: number, // between 0 and 1
    b: number, // between 0 and 1
    a: number, // between 0 and 1
  },
  points:[{ x: number, y: number, z: number }],
  id: number,
}
```
