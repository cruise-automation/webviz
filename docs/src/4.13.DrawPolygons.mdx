# DrawPolygons

`<DrawPolygons />`, combined with `PolygonBuilder`, allows drawing polygons on the X-Y plane. The example below shows how to combine the two to create a drawing tool. This tool works best when the user's camera is in non-perspective mode (2D).

## Props

| Name       | Type     | Default | Description                     |
| ---------- | -------- | ------- | ------------------------------- |
| `children` | `[]Polygon` | []   | Polygon class instances from DrawPolygons. |
